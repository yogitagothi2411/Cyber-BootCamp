# ATM Simulator

## Features
- PIN Authentication
- Menu-driven interface
- Withdraw Money
- Deposit Money
- Balance Inquiry
- Transaction Receipt
- Running Menu (while)
- Withdrawal Confirmation
- Total Deposited Tracking
- Total Withdrawn Tracking
- Date & Time
- Clean console formatting

## Concepts Used
- Variables
- if/else
- while loop
- match-case
- f-strings
- datetime module
- User input
- functions
